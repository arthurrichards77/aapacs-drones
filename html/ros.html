ROS

  1. Ensure the Xbee transciever (if you're using it) matches the drone you're using (7 = blue; 8=green)
  2. Launch your ROS activity, e.g. 
    roslaunch aapacs_drones quad_control_pelican_blue.launch
    or 
    roslaunch aapacs_drones quad_control_pelican_green.launch
  3. Check voltage reported similar to last battery test
  4. Check Vicon tracking reported OK e.g. 
    rostopic echo /vicon/PelicanBlue/PelicanBlue
    or 
    rostopic echo /vicon/PelicanGreen/PelicanGreen
    Check translation approx (0,0,0.2) and quaternion approx (0,0,0,1)
  5. Check drone comms: 
    rostopic echo /asctec/RC_DATA
    Wiggle transmitter sticks and confirm response via ROS.
